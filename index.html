What is crypto mining?

Crypto mining is the process of using powerful computers to solve complex mathematical puzzles that validate transactions on a blockchain. Once validated, these transactions are added to the blockchains public ledger. In return for securing the network and processing transactions, miners are rewarded with cryptocurrency.
